Contributions From Employee And Employer To Saudi Government

Cybrosys Technologies

Overview

This module is an application for handling the GOSI Contribution of Employees and Employers in Saudi. GOSI Contribution is only available for Saudi Citizens which has an age in between 18 and 60. This module also provides reports based on Contribution.

Features

  • ★   Created two new rules for employee and employer contributions
  • ★   Printing the pdf reports of contribution registers
